# Web

Lecca.io provides a Web tool that allows users to access the web using pre-built actions. This integration is designed to help you extract website content and perform Google searches efficiently. Please note that these actions use Credits.

  Note: These actions use Credits. Ensure you have sufficient Credits in your
  account to perform these actions.
</Callout>

## Actions

### Extract Website Content

The "Extract Website Content" action allows you to retrieve and extract content from a specified website. This can be useful for gathering information, data analysis, or content curation.

### Extract Static Website Content

The "Extract Static Website Content" only works on websites that load html and don't need Javascript to load data.

### Google Search

The "Google Search" action enables you to perform a search on Google and retrieve the results. This can be used for research, finding information quickly, or automating search tasks.

## Triggers

There are no triggers available for the Web integration at this time.

## Connections

There are no connections required for the Web integration.
