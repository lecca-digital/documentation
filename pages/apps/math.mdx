# Math - Math Helper

Lecca.io offers a Math Helper integration that allows users to perform basic mathematical operations within their workflows. This integration is perfect for automating calculations and enhancing the functionality of your AI Agents.

## Actions

### Evaluate Expression

Evaluate math expressions.

### Addition

The Addition action allows you to add two or more numbers together. Use this action to automate the process of summing values in your workflows.

### Subtraction

The Subtraction action enables you to subtract one number from another. This is useful for workflows that require calculating differences between values.

### Division

The Division action allows you to divide one number by another. Use this action to automate division operations in your workflows.

### Multiplication

The Multiplication action enables you to multiply two or more numbers. This action is ideal for workflows that require product calculations.

## Triggers

There are no triggers available for the Math Helper integration.

## Connections

There are no connections available for the Math Helper integration.
