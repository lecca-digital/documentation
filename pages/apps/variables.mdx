# Variables Integration

The Variables integration allows you to interact with your Lecca.io variables.

## Actions

### Get Variable

Get's a variable's properties using the ID of the variable.

### Update Variable

Updates a variable's data. Note that you can only update the data, not the data type. This means you cannot update a date on a number type.

### List Variables

Lists all variables avaialble within the project. This includes workspace and project variables.

### Create Knowledge Notebooks

Creates a new knowledge notebook. You can select whether to save the notebook to the project or workspace.

## Triggers

There are no triggers available for the Knowledge app integration.

## Connections

There are no connections available for the Knowledge app integration.
