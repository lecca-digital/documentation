# Google Drive Integration

Google Drive is a cloud-based storage service that enables users to store and access files online. This integration allows you to perform various actions on your files and folders stored in Google Drive, as well as set up triggers for new files.

## Actions

### Find File By Title

Search for a file in Google Drive by its title.

### Find Folder By Title

Search for a folder in Google Drive by its title.

### Export File

Export a file from Google Drive.

### Download File

Download a file from Google Drive to your local storage.

### Copy File

Create a duplicate of a selected file in Google Drive.

### Create Folder

Create a new folder in Google Drive.

### Share Folder

Share a specified folder with other users or entities via Google Drive.

### Share File

Share a specific file with other users or entities via Google Drive.

### Move File

Move a file from one folder to another within Google Drive.

### Move Folder

Move a folder to a different directory within Google Drive.

### Delete File

Delete a specified file from Google Drive.

## Triggers

### New File

Trigger an action when a new file is added to Google Drive.

### New File In Folder

Trigger an action when a new file is added to a specific folder in Google Drive.

## Connections

This integration uses OAuth2 for authentication. Ensure you have connected your Google Drive account and granted necessary permissions to enable these actions and triggers.
