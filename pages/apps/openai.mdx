# OpenAI Integration

OpenAI is a powerful AI model that can be used to generate human-like text and images. This integration allows you to leverage OpenAI's capabilities within your workflows on Lecca.io.

## Actions

### Chat from Text

The "Chat from Text" action allows you to input text and receive a response generated by OpenAI's language model.

## Triggers

There are no triggers available for the OpenAI integration at this time.

## Connections

To connect to OpenAI, you will need an API key. This key allows you to authenticate and interact with the OpenAI API securely. Make sure to keep your API key confidential and only use it in secure environments.
