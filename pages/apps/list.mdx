# List Helper Integration

List Helper is a List (Array) helper offered by Lecca.io. This integration provides a set of actions to manipulate and interact with lists efficiently.

## Actions

### Find Item In List

This action allows you to search for a specific item within a list. It returns the item if found.

### Combine Lists

This action enables you to merge two or more lists into a single list.

### Get Last Item In List

This action retrieves the last item from a list.

### Get First Item In List

This action retrieves the first item from a list.

### Count List

This action counts the number of items present in a list.

## Triggers

There are no triggers available for the List Helper integration.

## Connections

There are no connections available for the List Helper integration.
