# Knowledge Integration

The Knowledge allows you to utilize Lecca.io knowledge notebooks to store and retrieve information efficiently.

<Callout>This step uses Credits</Callout>

## Actions

### Search Knowledge

The "Search Knowledge" action enables you to search within your selected knowledge notebook, allowing you to quickly find the information you need.

### Save to Knowledge

Saves new text data to a knowledge notebook. You can only select notebooks available within the project.

### List Knowledge Notebooks

Lists all knowledge notebooks avaialble within the project. This includes workspace and project notebooks.

### Create Knowledge Notebooks

Creates a new knowledge notebook. You can select whether to save the notebook to the project or workspace.

## Triggers

There are no triggers available for the Knowledge app integration.

## Connections

There are no connections available for the Knowledge app integration.
