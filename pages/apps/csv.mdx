The CSV Helper app integration is a tool provided by Lecca.io that allows you to work efficiently with CSV data, enabling seamless data manipulation and transformation.

## Actions

### Convert CSV to JSON

The "Convert CSV to JSON" action allows you to transform your CSV data into JSON format. This action is useful for leveraging the flexibility and structure of JSON for various applications and workflows.
