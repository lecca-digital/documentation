# Google Docs Integration

Google Docs is a versatile online document editor that allows users to create, edit, and collaborate on documents in real-time. With its integration into Lecca.io, users can enhance their workflows by automating various document management tasks.

## Actions

### Create Document from Template

Create a new document using a predefined template in Google Docs.

### Create Document

Create a new blank document in Google Docs.

### Edit Template Document

Edit an existing template document, useful for updating template content without altering the original design.

### Append to Document

Add new content to an existing Google Doc, ideal for updating documents with new information.

### Find Document by Title

Search and locate a document in your Google Docs by its title.

### Find Document by Content

Search for a document based on its content within your Google Docs.

### Get Document Text

Retrieve the text from a specified Google Doc for review or use in another application.

## Triggers

### New Document

Trigger a workflow whenever a new Google Doc is created in your account.

### New Document In Folder

Trigger an action when a new document is created within a specific folder in Google Docs.

## Connections

### OAuth2

Connect your Google Docs to Lecca.io using OAuth2 for secure and seamless integration.
