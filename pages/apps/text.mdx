# Text Helper

Text Helper is a tool offered by Lecca.io that provides various text manipulation actions to enhance your workflows. Below are the available actions and information about triggers and connections.

## Actions

### Concatenate

Combine multiple text strings into one. This action is useful when you need to merge different pieces of text into a single string.

### Replace

Find and replace specific text within a string. This action allows you to substitute parts of your text with new content, making it easy to update or correct information.

### Search

Locate specific text within a string. This action helps you identify the presence and position of a substring within a larger text.

### Split

Divide a text string into multiple parts based on a specified delimiter. This action is ideal for breaking down text into manageable pieces, such as separating words or phrases.

## Triggers

There are no triggers available for the Text Helper integration.

## Connections

There are no connections available for the Text Helper integration.
