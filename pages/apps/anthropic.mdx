# Anthropic Integration

Anthropic provides powerfuls AI models that can be used to generate human-like text.

## Actions

### Chat from Text

The "Chat from Text" action allows you to input text and receive a response generated by the language model.

## Triggers

There are no triggers available for this integration at this time.

## Connections

To connect to Anthropic, you will need an API key. This key allows you to authenticate and interact with the Anthropic API securely.
