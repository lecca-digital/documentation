# Calendly Integration

Calendly is a modern scheduling platform that simplifies the process of booking meetings and events. With this integration, you can automate your scheduling workflows and enhance productivity.

## Actions

### List User Events

Retrieve a list of events scheduled by a specific user. This action allows you to access and manage user-specific events efficiently.

### List Organization Events

Fetch a list of events scheduled within an organization. This action helps in overseeing and coordinating events across the entire organization.

## Triggers

There are no triggers available for the Calendly integration at this time.

## Connections

Calendly connects using OAuth2, ensuring secure and seamless integration with your scheduling workflows.
