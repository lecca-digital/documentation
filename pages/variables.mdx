# Variables

You can create a **variable** in Lecca.io to use within workflow actions. This is useful for referencing common values like names, emails, phrases, dates, numbers, and more as a variable.

## Types of Variables

There are four types of variables you can create:
- **String**: Text values.
- **Number**: Numeric values.
- **Boolean**: True/False values.
- **Date**: Date values.

## Creating Variables

Variables can be created at both the **workspace** and **project** levels.

<Steps>
### Go to the Variables page

Navigate to the **Variables** page to create a new variable.

### Open the New Variable dialog

Click on **New Variable**.

### Select the variable type

Select the **variable type**: String, Number, Boolean, or Date.

### Enter value

Enter the variable details.

### Save 

Save your variable.
